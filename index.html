LivingWord

Bible Reading Component

View project on GitHub





NEW! Version 3.0.0 for Joomla 3.x!! Version 2.5.0 is still available for Joomla 1.5x-2.5x. LivingWord is a Joomla bible reading component. Utilizing bible resources from BibleGateway.com, this component offers many bible reading plans, bible versions and translations, and audio versions as well. With the use of an included plugin, users can have their daily readings emailed to them. A daily bible reading module is also included.




Features include:

  • Bible translations in many different languages
  • Several different Bible reading plans
  • Capability to create your own reading plan through the Administrative console
  • Bible resources include non-English Bibles available on PDF
  • Bible study tools include commentaries, Bible search, and keyword search
